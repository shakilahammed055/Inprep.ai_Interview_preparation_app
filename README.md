# Inprep_AI

Inprep_AI is a mobile application designed to help users prepare for job interviews and track their job applications. It provides personalized interview practice, AI-powered feedback, progress tracking, and a user-friendly interface to manage profiles and notifications. The design is built using Figma, with assets and specifications available for development using Flutter.



## Features

- **Interview Progress Tracking**: Monitor interview performance over time with weekly insights.
- **AI-Powered Feedback**: Receive detailed feedback on content accuracy, articulation clarity, and more.
- **Personalized Practice**: Access job-specific questions and unlimited mock interviews.
- **Profile Setup**: Configure personal information, experience, education, and subscription plans.
- **Job Application Management**: Track applied jobs and schedule mock interviews.
- **Notifications**: Stay updated with task alerts and interview reminders.
- **Subscription Plans**: Choose from free, premium, or pay-per-interview options.

## Getting Started

### Prerequisites

To get started with the development of the InterviewPrep app, ensure you have the following:

- Flutter SDK (install from [flutter.dev](https://flutter.dev))
- Dart (included with Flutter)
- Android Studio or Xcode (for Android/iOS emulation)

   ```bash

3. **Install Dependencies**:

   ```bash
   flutter pub get
   
4. **Run the Application**:

    ```bash

## Usage

Once the app is running, you can:

- **Log in or sign up** using the authentication screens.

- **Set up your profile** with personal details and preferences.

- **Track interview** progress and apply for jobs via the "My Jobs" section.

- **Use mock interviews** and review AI feedback in the "Interview" section.

- **Download progress reports** from the "Progress & Insights" screen.
